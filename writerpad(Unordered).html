WritePad Editor

WritePad is an editor application that allows you to creat Web easily and quickly.

Step to open WritePad application:

Click Start
Click Programs
Click WritePad

Advantages of using a WritePad editor:

It allows faster creation of Web pages.
It allows easier updation of text or images in WritePad editor.
It allows platform independent.

Disadvantages of using a WritePad editor:

It consumes more memory space.
It requires ti fill-in more details such as meta data for each page.
